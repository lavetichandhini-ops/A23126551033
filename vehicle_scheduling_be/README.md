# Vehicle Maintenance Scheduler

## Problem
Select maintenance tasks to maximize total operational impact without exceeding available mechanic hours.

## Approach
Used Dynamic Programming (0/1 Knapsack).

- MechanicHours -> Capacity
- Duration -> Weight
- Impact -> Value

## Output

Depot 1 -> Maximum Impact = 109

Depot 2 -> Maximum Impact = 169

Depot 3 -> Maximum Impact = 181

Depot 4 -> Maximum Impact = 143
